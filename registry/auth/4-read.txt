
Pour le moment, nous allons mettre en place une Authentification de base avec un utilisateur et un mot de passe globaux Ã  l'aide du fichier htpasswd. 


mkdir auth, dossier dans lequel on genere un htpasswd


generer notre fichier htpasswd
docker run --rm \
    --entrypoint htpasswd \
    registry:2 -Bbn testuser testpassword > auth/htpasswd


demarrer notre container en prenant en compte auth/htpasswd

docker run -d \
    -p 5000:5000 \
    --restart=always \
    --name mon-registry \
    -v "$(pwd)"/data:/var/lib/registry \
    -v "$(pwd)"/auth:/auth \
    -e "REGISTRY_AUTH=htpasswd" \
    -e "REGISTRY_AUTH_HTPASSWD_REALM=Registry Realm" \
    -e REGISTRY_AUTH_HTPASSWD_PATH=/auth/htpasswd \
    -v "$(pwd)"/certs:/certs \
    -e REGISTRY_HTTP_TLS_CERTIFICATE=/certs/localhost.crt \
    -e REGISTRY_HTTP_TLS_KEY=/certs/localhost.key \
    registry:2


se logger
=> docker login localhost:5000
=> Username: testuser
=> Password: testpassword

si tout est ok, on a ===> Login Succeeded

on peut pousser des images docker push localhost:5000/alpinemysql
